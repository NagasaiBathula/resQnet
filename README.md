# ResQNet AI

Coordinated Emergency Response and Resource Dispatch Hub.

## Development Startup

To run the complete stack (both the TanStack Start React frontend and Express/MongoDB backend) simultaneously, execute the following commands from the root directory:

```bash
npm install
npm run dev
```

This starts both services and prefixes console outputs cleanly:
- `[frontend]` - Vite development server (port 8080)
- `[backend]` - Express server and MongoDB connector (port 5000)

No separate terminal operations are required.
